# SPK SAW Pemilihan Laptop Terbaik

Aplikasi Sistem Pendukung Keputusan menggunakan metode **Simple Additive Weighting (SAW)** untuk menentukan laptop terbaik berdasarkan beberapa kriteria.

## Kriteria yang Digunakan
- Harga (Cost)
- RAM (Benefit)
- SSD (Benefit)
- CPU / Performa (Benefit)
- Berat (Cost)

## Fitur Aplikasi
- Input alternatif laptop
- Input bobot kriteria
- Input penilaian laptop
- Perhitungan SAW (Matrix X, Normalisasi, Nilai Preferensi)
- Hasil perankingan laptop terbaik

## Teknologi
- PHP Native
- MySQL
- Bootstrap
- XAMPP

## Cara Menjalankan
1. Letakkan folder project ke `htdocs`.
2. Buat database (misal: `spksaw`).
3. Import file SQL.
